## Setup

### `plugins`

```js
import { Icons } from '@vime-js/player';

player.plugins = [Icons];
```

### `PluginsManager`
```js
import { Icons } from '@vime-js/player';

player
  .getPluginsManager()
  .addPlugin(Icons)
  .then((icons) => {
    // ...
  });
```
