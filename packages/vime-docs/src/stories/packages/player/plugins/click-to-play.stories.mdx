# ClickToPlay `Plugin`

[View Source](https://github.com/vime-js/vime/blob/master/packages/vime-player/src/plugins/ClickToPlay.svelte)

**ID:** `vClickToPlay`

This plugin adds the ability to click the player to toggle playback.

## Setup

### `plugins`

```js
import { ClickToPlay } from '@vime-js/player';

player.plugins = [ClickToPlay];
```

### `PluginsManager`
```js
import { ClickToPlay } from '@vime-js/player';

player
  .getPluginsManager()
  .addPlugin(ClickToPlay)
  .then((clickToPlay) => {
    // ...
  });
```

## Props

### `autopilot: boolean`

**Default:** `true`

In autopilot mode the plugin will control certain properties automatically. Set this to `false` if you'd like to 
control them yourself. Properties marked with `auto` are part of this system.

### `isEnabled: boolean` `(auto)`

**Default:** `true`

Whether the plugin is enabled or not.
