# ToggleControl `Component`

[View Source](https://github.com/vime-js/vime/blob/master/packages/vime-player/src/plugins/controls/control/ToggleControl.svelte)

A `Control` (see <LinkTo kind="packages-player-plugins-controls-control-control--page">Controls/Control</LinkTo>)
which toggles between an active and inactive state when clicked.

## Props

### `label: string|undefined`

**Default:** `undefined`

Forwarded to the `Control` component.

### `activeTitle: string|undefined`

**Default:** `undefined`

The `Control` title prop when the control is in an active state.

### `inactiveTitle: string|undefined`

**Default:** `undefined`

The `Control` title prop when the control is in an inactive state.

### `activeIcon: string|undefined`

**Default:** `undefined`

The icon to use when the control is in an active state.

### `inactiveIcon: string|undefined`

**Default:** `undefined`

The icon to use when the control is in an inactive state.

### `isEnabled: boolean`

**Default:** `true`

Whether to render the control in the DOM or not.

### `isActive: boolean`

**Default:** `false`

Whether the control is in an active state or not.

### `large: boolean`

**Default:** `false`

Forwarded to the `Control` component.

## Methods

### `getControl(): Control`

The underlying `Control` (see <LinkTo kind="packages-player-plugins-controls-control-control--page">Controls/Control</LinkTo>) instance.

## Events

Forwards all `Control` events (see <LinkTo kind="packages-player-plugins-controls-control-control--page#events">Controls/Control - Events</LinkTo>).
