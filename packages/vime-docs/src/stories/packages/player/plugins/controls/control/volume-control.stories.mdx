# VolumeControl `Component`

**LABEL:** `adjustVolume`

A control that adjusts the volume level of the player.

## Methods

### `getEl(): HTMLElement`

The root HTML element of the component.

### `getSlider(): <input type="range">`

The [input range slider](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input/range) that
controls the `volume` of the player.

### `getMuteControl(): MuteControl`

The underlying `MuteControl` (see <LinkTo kind="packages-player-plugins-controls-control-mutecontrol--page">Controls/MuteControl</LinkTo>) instance.
