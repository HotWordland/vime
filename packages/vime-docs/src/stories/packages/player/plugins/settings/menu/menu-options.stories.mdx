# MenuOptions `Component`

[View Source](https://github.com/vime-js/vime/blob/master/packages/vime-player/src/plugins/settings/menu/MenuOptions.svelte)

Groups and renders multiple `MenuItemRadio` (see <LinkTo kind="packages-player-plugins-settings-menu-menuitemradio--page">Menu/MenuItemRadio</LinkTo>) 
components.

## Props

### `group: any`

**Default:** `undefined`

The value of the currently selected radio button.

### `options: any[]`

**Default:** `[]`

List of objects, where each `object` is a set of `MenuItemRadio` props (see <LinkTo kind="packages-player-plugins-settings-menu-menuitemradio--page#props">Menu/MenuItemRadio - Props</LinkTo>).

## Methods

### `getRadios(): MenuItemRadio[]`

The rendered `MenuItemRadio` instances.

## Events

### `valuechange`

**Data Type:** `any`

Fired when a radio button is clicked. The event data contains the radio's respective value.
