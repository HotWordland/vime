# Scrim `Plugin`

[View Source](https://github.com/vime-js/vime/blob/master/packages/vime-player/src/plugins/Scrim.svelte)

**ID:** `vScrim`

A scrim is a temporary low opacity black surface for making controls or other content more visible.

ðŸ’¡ To see this plugin in action then go to 'Player/Components'.

## Setup

### `plugins`

```js
import { Scrim } from '@vime-js/player';

player.plugins = [Scrim];
```

### `PluginsManager`
```js
import { Scrim } from '@vime-js/player';

player
  .getPluginsManager()
  .addPlugin(Scrim)
  .then((scrim) => {
    // ...
  });
```

## Props

### `autopilot: boolean`

**Default:** `true`

In autopilot mode the plugin will control certain properties automatically. Set this to `false` if you'd like to 
control them yourself. Properties marked with `auto` are part of this system.

### `isEnabled: boolean` `(auto)`

**Default:** `true`

Whether the plugin is enabled or not.

### `isActive: boolean` `(auto)`

**Default:** `false`

Whether the scrim is visible or not.
