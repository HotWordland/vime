# KeyboardShortcut `interface`

A keyboard shortcut is an `object` with the following properties:

### `hint: string` 

Text that helps the user understand what key to press to use this shortcut.

### `keys: int[]` 

The [JS key codes](https://keycode.info) to listen to.

### `action: () => void` 

A callback to call when the key is pressed.
