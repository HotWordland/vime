# DblClickFullscreen `Plugin`

[View Source](https://github.com/vime-js/vime/blob/master/packages/vime-player/src/plugins/DblClickFullscreen.svelte)

**ID:** `vDblClickFullscreen`

This plugin adds the ability to double click the player to toggle fullscreen mode.

## Setup

### `plugins`

```js
import { DblClickFullscreen } from '@vime-js/player';

player.plugins = [DblClickFullscreen];
```

### `PluginsManager`
```js
import { DblClickFullscreen } from '@vime-js/player';

player
  .getPluginsManager()
  .addPlugin(DblClickFullscreen)
  .then((dblClickFullscreen) => {
    // ...
  });
```
