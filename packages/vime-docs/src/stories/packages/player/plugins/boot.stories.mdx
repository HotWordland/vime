## Setup

```js
import { Boot } from '@vime-js/player';

player.plugins = [Boot];
```

## Props

### `manifest: { [id]: boolean }`

**Default:** All plugins default to `true`.

The manifest determines which plugins should be installed. All plugins are identified by
their `ID` with the `v` prefix removed and first letter lowercased.

**Examples**

- `vControls` = `controls`
- `vActionDisplay` = `actionDisplay`
- `vClickToPlay` = `clickToPlay`

```js
// ...

player.$on('mount', () => {
  // Turn off these plugins.
  player.vBoot.manifest = {
    controls: false,
    actionDisplay: false,
    clickToPlay: false
  };
});
```
