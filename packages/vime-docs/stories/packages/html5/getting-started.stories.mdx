## Installation

**NPM:** `npm install @vime/html5`

**YARN:** `yarn add @vime/html5`

**UNPKG:** `https://unpkg.com/@vime/html5`

The `dist` folder inside the package contains multiple exports:

- `html5.js` is a UMD development build that can be used directly in the browser via a `<script>` tag.
- `html5.esm.js` is intended for use with modern bundlers like [Webpack](https://webpack.js.org/) or [Rollup](http://rollupjs.org/guide/en/).
- `html5.min.js` is a UMD production build that can be used directly in the browser via a `<script>` tag.
- `html5.esm.min.js` can be used directly in modern browsers via `<script type="module">` tag.

üìù **_If you're using a bundler then it will automatically pull in the correct files._**

üìù **_If you're using [svelte-loader](https://github.com/sveltejs/svelte-loader) or [rollup-plugin-svelte](https://github.com/sveltejs/rollup-plugin-svelte) then you'll receive the uncompiled components._**

## Setup

### JavaScript

üìù **_See the Svelte [Client-side component API](https://svelte.dev/docs#Client-side_component_API)._**

```js
import { Html5 } from '@vime/html5';

// Mount
const target = document.getElementById('player-target');
const player = new Html5({ target });

// ...

// Destroy
player.$destroy();
```

### Svelte

```html
<Html5 />

<script>
  import { Html5 } from '@vime/html5';
</script>
```

## Usage

üìù **_This component returns the core Vime player, see <LinkTo kind="api-player--page">API/Player</LinkTo>.
Keep reading for how certain properties behave with Html 5._**

### src

```js
// Null
player.src = null;

// Relative
player.src = '/media/audio.mp3';

// Absolute
player.src= 'https://mysite.com/video.mp4';

// Object
player.src = { src: '/media/video.mp4', type: 'video/mp4' };

// Array of strings
player.src = [
  '/media/video.mp4',
  '/media/video.ogg'
];

// Array of objects
player.src = [{ 
  src: '/media/video.mp4', 
  type: 'video/mp4' 
}, { 
  src: '/media/video.ogg', 
  type: 'video/ogg' 
}];
```

### quality

Qualities can be set for videos and changed programatically. The initial quality is set based on the
aspect ratio and width of the player.

```js
// Passing in qualities
player.src = [{ 
  src: '/media/video-1080.mp4', 
  type: 'video/mp4' 
  quality: 1080
}, { 
  src: '/media/video-1080.ogg', 
  type: 'video/ogg',
  quality: 1080
}, { 
  src: '/media/video-720.mp4', 
  type: 'video/mp4' 
  quality: 720
}, { 
  src: '/media/video-720.ogg', 
  type: 'video/ogg',
  quality: 720
}];

// Getting current quality
const currentQuality = player.quality;

// Setting a quality
player.quality = 720;
```
